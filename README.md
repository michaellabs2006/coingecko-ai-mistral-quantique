CoinGecko AI Mistral Quantique is a compact web application for crypto market analysis tasks. It collects live market information from CoinGecko and then applies Mistral-based AI guidance to help interpret both individual assets and larger coin groups.

It is intended for users who want a focused dashboard for quicker comparison, review, and decision support without introducing a bulky runtime. Built around HTML, PHP, and JavaScript, it stays straightforward to deploy and practical for routine use.
